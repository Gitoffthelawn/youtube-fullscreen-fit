Custom icon position

On YouTube Mobile, the icon is usually displayed at the top-left corner. If this interferes with other extension, you can change the position here.
Note: you must use CSS values there, for example:

  • If you want to put the icon at the top, write something like 10px in the "Top" textbox
  • If you want to put the icon at the bottom, write something like calc(100vh - 40px - 10px) in the "Top" textbox
  • If you want to put the icon at the left, write something like 10px in the "Left" textbox
  • If you want to put the icon at the right, write something like calc(100vw - 40px - 10px) in the "Left" textbox

Note: the icon background is a square of 40px. So, when you use calc(), remember to subtract at least 40 pixels so that it can be shown!
